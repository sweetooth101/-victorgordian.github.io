Why Git?

Git is a powerful version control tool used in modern software development to manage code history, support teamwork, and maintain multiple versions of a project. By storing changes locally and remotely, it ensures that work is never lost and can easily be shared or rolled back.

Common commands

  • git clone <url> – copy a remote repo
  • git checkout -b feature/login – new branch
  • git add . → git commit -m "msg" → git push
  • git pull – sync from remote
  • git merge main – merge another branch

Branch strategy

Keep main releasable, use short-lived feature branches, and open pull requests with code review.
